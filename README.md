# vue3-scroll-number
一个仅支持 vue3 的滚动数字组件

## 安装
```
npm install vue3-scroll-number
```
或者
```
pnpm add vue3-scroll-number
```

## 使用
```
import Vue3ScrollNumber from 'vue3-scroll-number/components/Vue3ScrollNumber.vue'
 ```
```
 <Vue3ScrollNumber
      style="font-size: 55px;"
      :from="21"
      :to="value"
    />
```
